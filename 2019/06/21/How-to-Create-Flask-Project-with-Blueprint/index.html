How to Create Flask Project with Blueprint

Posted on 2019-06-21 | In manual

Before we set off

We can create two types of architecture based on Flask and Blueprint. One of the 2 is Functional Structure that you can organize the pieces of app by what the do. Templates are grouped together in one directory, static files in another and views in a third.

And the next one is Divisional, with the divisional structure, you organize the pieces of the application based on which part of the app they contribute to. All of the templates, views and static files for the admin panel go in one directory, and those for the user control panel go in another. Here are what they look like:
